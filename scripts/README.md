# Scripts

This directory contains utility scripts for development and maintenance of the English Tests project.

## generate_text.py

A script for generating B1-level English text using AI models via the Horde SDK.

### Usage

```bash
pip install -e ".[experimental]"
python scripts/generate_text.py --apikey "your-api-key"
```

### Requirements

Install experimental dependencies:

```bash
pip install -e ".[experimental]"
```

Or manually:

```bash
pip install horde_sdk
```
